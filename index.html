This shit is arcane

So far the hardest thing to learn is how each element fits with the surrounding/enclosing elements. The developer tools have been critical in figuring out what formatting is coming from where.

The lecture Tuesday night about how floating elements are just a completely different universe than non-floated elements was extremely helpful. Who would have guessed it was so funky?

Looking forward to what's next

I could make this prettier but I'm not going to

I feel like to get more out of this exercise I would start from scratch and rewrite it. But I'd rather start with something new, so am looking forward to class tonight for our next project.

I also think I have a pretty good idea of how git works, and ditto on looking forward to getting to use it for a specific project.

The ping pong exercise was good but it would have helped to have a little context for how git usually fits into a development process. We kept trying things and ending up with weird results, but they were things we'd never actually do.

But then that's how you want us to learn :)

The demo of Chrome developer tools was really good - I'd been using it but hadn't gone exploring into different tabs.
